# アプリケーション名
問い合わせフォーム

## Dockerビルド
１,git clone 
２，docker-compose up -d --build

## 環境構築
１，docker-copose exe php bash
２，composer install
３，.env.exampleファイルから.envを作成して環境変数を変更
４，php artisan key:generate
５，php artisan migrate
６，php artisan db:seed

## 使用技術(実行環境)
